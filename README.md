# Football Player Stats Analysis 2024/2025 ⚽📊

This project is a Power BI dashboard for analyzing football player statistics from the 2024/2025 season. The dashboard visualizes player performance using goals, assists, expected goals (xG), league, club, position, and playing time.

## Objectives

- Analyze football player performance
- Compare top players by goals and xG
- Create an interactive Power BI dashboard
- Practice data analysis and business intelligence skills

## Tools

- Power BI
- Power Query
- CSV dataset
- Data Visualization

## Dashboard Features

- Total player KPI card
- League slicer
- Player detail table
- Top goalscorers chart
- Top xG players chart
- Interactive filtering

## Key Questions

- Who scored the most goals?
- Who had the highest expected goals?
- Which players performed best across leagues?
- How can football statistics be visualized for analysis?
